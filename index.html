Welkom op deze one-pager van mijn PLE project

Wat is het?

Heb je wel eens rondgekeken om een nieuwe computer te kopen maar waren deze te duur? Je kan zelf van losse onderdelen best makkelijk een computer bouwen wat een stuk goedkoper is dan er een kopen. Het kan best intimiderend zijn om zelf een computer van losse onderdelen te bouwen. Dit product helpt je met het zelf bouwen van een computer.

Hoe werkt het?

Als je onderdelen op een webshop bestelt of een pc hebt samengesteld op PcPartPicker krijg je een Qr-code waarmee je de onderdelen op de app kan inladen. Als deze zijn ingeladen kan je met je camera naar een plat ondergrond kijken en de pc neerzetten.

Als je een model hebt geplaats kan je kiezen wat je wilt doen. Als je niet weet waar je moet beginnen kan je een stppenplan doorlopen die je in de meest optimale volgorde laat zien hoe je de onderdelen moet instaleren. De uitleg van wat je moet doen krijg je in de vorm van animaties die laten zien wat je moet doen. Je krijgt bij elke stap ook extra tips en tricks van waar je op moet letten en wat veelgemaakte fouten zijn.

Als je wat meer ervaren bent kan je er ook voor kiezen om zelf je volgorde te kiezen waarin je de onderdelen installeert.

Inspiratie

Als je nog niet weet wat je voor onderdelen wilt hebben voor je computer kan je ook naar de builds van andere kijken. Als mensen een build hebben gemaakt kunnen zij die uploaden en komen ze in een publiek overzicht te staan. Op dit overzicht kan je kijken wat andere mensen voor combinaties hebben gekozen voor hun build en hier inspiratie uit halen

Als je zelf een build hebt gemaakt kan je deze ook uploaden. Je kan hier dan reacties op krijgen zodat andere mensen kunnen laten weten wat zij er van vinden en wat zij mischien anders zouden doen.
